# ParkFlow

ParkFlow is a parking rental marketplace for India.

## Current Status

This folder contains the initial scaffold for the ParkFlow monorepo:

- `client/` for the React frontend
- `server/` for the Node.js/Express backend
- shared environment and deployment docs

The full product build will be added in phases so the API and UI contracts stay stable.

## Local Setup

```bash
cd /Users/siyran/Documents/Python/ParkFlow
npm install
npm run dev
```

## Structure

- `client/` contains the frontend scaffold.
- `server/` contains the backend scaffold.
- `server/database/` contains the SQL schema and demo seed data.
